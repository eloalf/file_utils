# file_utils

A small collection of useful tools mostly for LINUX to assist daily work with various kinds of data files and formats.

### bhist.c
Written in C under LINUX.<br/>
Sums up all equal byte values into an array of 256 bins, then builds, stores and launches a gnuplot script for logarithmic display of the histogram data. Requires an installation of gnuplot. Look up comment lines in source code for more information.
